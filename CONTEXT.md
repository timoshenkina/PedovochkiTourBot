# Pedovki Tour — Telegram Mini App
> Контекст проекта для Claude Code. Прочитай этот файл и index.html перед любыми изменениями.

---

## Что это за проект

Telegram Mini App — личный гид по Львову. Пользователь открывает бота, выбирает места (кафе, парки, музеи, фотолокации), добавляет в план дня (Початок / Середина / Кінець дня) и сохраняет в избранное. Всё сохраняется в Telegram CloudStorage — привязано к аккаунту пользователя.

---

## Технический стек

- **Фронтенд:** чистый HTML/CSS/JS — один файл `index.html`
- **Хостинг:** Netlify → `https://effortless-crisp-69ff24.netlify.app`
- **Telegram бот:** `@PedovockiTourBot`
- **Mini App ссылка:** `t.me/PedovockiTourBot/places`
- **Данные:** Google Sheets → Google Apps Script (JSON API)
- **Хранилище юзера:** Telegram CloudStorage (встроено в Telegram WebApp SDK)

**Структура Google Sheets** (лист называется `Places ` — с пробелом в конце!):
| Колонка | Поле | Пример |
|---------|------|--------|
| A | id | "1 погуляти" |
| B | name | "Парк Івана Франка" |
| C | category | "park" |
| D | address | "вул. Університетська, 1" |
| E | price | "$" |
| F | map_url | "Ivan Franko Park" (название для Google Maps) |
| G | rating | число (в Excel хранится как дата — баг) |
| H | open_hours | "08:00 - 22:00" |
| I | description | текст |
| J | photo_url | прямая ссылка на фото Google Maps |

**Все категории в таблице:**
- `park` → Погуляти 🌿
- `cafe` → Поїсти 🍽️
- `museum` → Подивитися 🖼️
- `photo-place` → Пофоткатися 📸
- `cafe/ photo-place` → смешанная (нормализуется через `normCat()`)

**Важно:** категории нормализуются функцией `normCat()` в JS — она приводит все варианты к `park/cafe/museum/photo`.

## Структура приложения

### Экраны (3 штуки, переключаются через `showScreen()`):

**1. Головна (`screenHome`)**
- Заголовок "Каталог місць"
- 4 карточки категорий: Поїсти / Погуляти / Подивитися / Пофоткатися
- При клике → переход в Каталог с нужным фильтром
- Раздел "Мої фаворити для туру" — 3 раздела по 3 слота каждый:
  - Початок дня (homeFavRow0)
  - Середина дня (homeFavRow1)
  - Кінець дня (homeFavRow2)

**2. Каталог (`screenCatalog`)**
- Фильтры категорий: Всі / Поїсти / Погуляти / Подивитися / Пофоткатися
- Подфильтры: Все / Поруч / Нові / Улюблені / Тур
- Список мест с фото (или цветной блок если фото нет)
- Кнопки ➕ (добавить в тур) и 🤍 (в избранное) на каждой карточке
- Внизу: "Мої фаворити для туру" с первым рядом слотов

**3. Улюблені (`screenFavs`)**
- Список сохранённых мест
- Кнопка "Всі oo улюблених"
- 3 раздела тура: Початок / Середина / Кінець дня

### Bottom Nav (3 кнопки):
- 🏠 Головна
- 🗺️ Каталог
- ❤️ Улюблені

### Детальная страница (modal bottom sheet `detailOverlay`):
- Большое фото вверху (или цветной блок с эмодзи)
- Название, рейтинг, цена
- Адрес, часы работы, соцсети
- Описание
- 3 кнопки: ➕ в тур / ❤️ в избранное / ✕ закрыть

## CloudStorage (сохранение данных юзера)

Используется `window.Telegram.WebApp.CloudStorage` — данные привязаны к Telegram аккаунту пользователя, синхронизируются между устройствами.

**Ключ:** `pedovki_data`

**Что сохраняется:**
```json
{
  "favs": [0, 3, 7],
  "tourPlaces": [[1, 4], [2], [5, 8, 9]]
}
```

**Логика:**
- `loadFromCloud(callback)` — вызывается при запуске, до загрузки мест
- `saveToCloud()` — вызывается автоматически при каждом изменении (❤️, ➕, удаление из тура)
